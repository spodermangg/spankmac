# SpankMac

For Apple Silicon MacBooks. Detect physical hits using the internal accelerometer and play audio. [Based on taigrr/spank](https://github.com/taigrr/spank), added custom features.

## Prerequisites

- macOS on Apple Silicon (M-series)
- `sudo` access

## Quick Start

```bash
Download latest release.  
Drag into Applications folder.  
Run the Application and HF.  
```

## Modes

- **Default**: Random pain/protest sounds.
- **Sexy**: Escalating intensity based on slap frequency.
- **Halo**: Halo death sound effects.
- **Lizard**: Escalating lizard-themed responses.
- **Half-Life 2**: Sounds from Half-Life 2.
